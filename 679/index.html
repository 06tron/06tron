This page uses four optional query parameters to allow for users to add a custom function. They are: the function body 'f', the function's title in the select element 'f-name', the default value of the function argument 'a', and a comment on the argument's purpose 'a-hint'.

String.prototype.userFunction = function (arg, fellback) {
	/* contents of 'f' parameter */
}

The arguments of the custom function are the string 'arg' and the boolean 'fellback'. The former is the optional parameter passed through the above input element, and the latter is true as long as the input is blank and the default value was passed instead. Examples: To Uppercase, Erase All, and Add Dots.
